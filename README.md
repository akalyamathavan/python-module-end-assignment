# python-module-end-assignment

**Project Description :**
Customer support teams receive many service tickets every day.
This project is a Python-based Customer Support Ticket Analysis System that helps store, clean, analyze, and extract insights from customer support tickets to identify common issues, customer sentiment, and priority trends.

**Project Objectives :**
Store and manage customer support tickets
Allow users to add new tickets dynamically
Clean and normalize issue descriptions
Perform keyword-based issue analysis
Analyze ticket priorities
Generate meaningful insights for decision-making,

**Tech Stack**
Language: Python
Environment: Google Colab / Jupyter Notebook
Concepts Used:
Lists & Dictionaries
Loops & Conditional Statements
Functions
String Manipulation
Data Analysis.
